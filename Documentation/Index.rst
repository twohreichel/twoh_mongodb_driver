The **MongoDB Driver** extension adds native MongoDB support to TYPO3 v12.
It provides:

- A custom database driver for MongoDB
- A connection pool adapter for simple usage in controllers/services
- A backend module for browsing MongoDB collections
- A clean wrapper around MongoDB queries
- Console command for mass test data generation

This document provides installation, configuration, and usage instructions.
